# Drawly

A drawing and visualization web application built with JavaScript, CSS, and HTML. Drawly provides an intuitive interface for creating, sketching, and designing diagrams and artwork directly in your browser.

## Features

- 🎨 **Intuitive Drawing Tools** - Easy-to-use sketching and drawing capabilities
- 💾 **Save & Export** - Save your creations locally or export to various formats
- 🎯 **Precision Controls** - Fine-tuned drawing controls for detailed work
- ⚡ **Responsive Design** - Works seamlessly across different devices and screen sizes
- 🖱️ **Mouse & Touch Support** - Full support for both mouse and touch input

## Tech Stack

- **Frontend**: JavaScript (59.5%), CSS (25.1%), HTML (0.2%)
- **Scripts**: PowerShell (8.4%), Shell (6.8%)
- **Architecture**: Modular component-based design

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Git (for cloning the repository)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vansh7nvc/Drawly.git
cd Drawly
```

2. Open the application:
```bash
# Simply open the index.html file in your web browser
open index.html
```

Or use a local development server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Usage

1. **Start Drawing**: Use the drawing tools to create sketches
2. **Customize**: Adjust colors, brush sizes, and styles as needed
3. **Save**: Export your artwork when you're done
