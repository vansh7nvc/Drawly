                                                                                                                                                                # **Excalidraw: Comprehensive Feature List**

## **1\. Core Drawing & Canvas (Free & Open Source)**

The fundamental "hand-drawn" experience that makes Excalidraw unique.

* **Hand-Drawn Aesthetic:** Uses "Excalifont" and rough-stroke algorithms for an informal, non-intimidating sketching feel.  
* **Infinite Canvas:** An unbounded workspace that expands as you draw in any direction.  
* **Smart Binding:** Arrows and lines "bind" to shapes; moving the shape automatically adjusts the connector's length and angle.  
* **Elbow Arrows:** Orthogonal (90-degree) connectors that automatically route around objects for clean flowcharts.  
* **Shape Library Marketplace:** Access a community-driven repository of pre-made icons (AWS, GCP, Kubernetes, UI Kits, etc.).  
* **Advanced Element Switching:** Quickly toggle selected shapes between rectangles, ellipses, and diamonds using the Tab key.  
* **Precision Controls:** Adjustable grid snapping, stroke width, opacity, and fill styles (solid, cross-hatch, hachure).  
* **Image Handling:** Drag-and-drop image support with built-in cropping and layering tools.  
* **Dark Mode & Theming:** Native support for Dark/Light modes, including automatic system-theme matching.  
* **CJK Support:** Hand-drawn font styles specifically optimized for Chinese, Japanese, and Korean characters.

## **2\. Generative AI Capabilities**

Advanced tools that bridge the gap between sketching and production.

* **Text-to-Diagram:** Generate full flowcharts, mind maps, or system architectures from natural language prompts.  
* **Wireframe-to-Code:** Convert hand-drawn UI sketches into interactive React, Tailwind, or HTML code snippets.  
* **Mermaid-to-Excalidraw:** Paste Mermaid.js code to instantly render editable, hand-drawn diagrams.  
* **AI Refinement:** Use AI to "clean up" rough sketches or expand on a drawn concept with one click.  
* **Bring Your Own Key (BYOK):** (Plus feature) Ability to use your own OpenAI API key for higher usage limits.

## **3\. Collaboration & Security**

Features designed for live teamwork and data protection.

* **Real-Time Collaboration:** Share a live session link for multi-user editing with visible cursor tracking.  
* **End-to-End Encryption (E2EE):** Shared sessions are encrypted locally; the server never sees your actual drawing data.  
* **Voice Hangouts:** Integrated peer-to-peer voice chat directly within the whiteboard (Plus).  
* **Live Screensharing:** Share your screen with collaborators inside the drawing session (Plus).  
* **Comments & Feedback:** Attach sticky comments to specific shapes with emoji support for async reviews (Plus).  
* **Read-Only Links:** Generate permanent, non-editable URLs for documentation or sharing with clients.

## **4\. Presentation & Export**

Tools for moving from brainstorming to formal meetings.

* **Frame Tool:** Define specific areas of the canvas as "slides."  
* **Presentation Mode:** Navigate through frames with smooth animations and a laser pointer tool.  
* **QR Remote Control:** Scan a code to control your presentation slides from a mobile device.  
* **Multi-Format Export:** \* **Standard:** PNG, SVG, and .excalidraw (JSON).  
  * **Premium:** High-fidelity PDF and PPTX (PowerPoint) exports.  
* **Clipboard Integration:** Copy elements as PNGs directly to your clipboard for quick pasting into Slack or Notion.

## **5\. Professional & Power User Tools (Excalidraw+)**

Features tailored for long-term project management and teams.

* **Cloud Persistence:** Securely save drawings to a workspace instead of relying on local browser storage.  
* **Folder Organization:** Nested directory structures to manage hundreds of diagrams.  
* **Role-Based Access (RBAC):** Manage team permissions (Admin, Member, Guest) and workspace-wide sharing settings.  
* **Full-Text Search:** Search for specific text across all drawings in your entire cloud workspace.  
* **Command Palette:** Access all editor tools and app commands via Cmd/Ctrl \+ / for a keyboard-first workflow.  
* **SOC 2 Compliance:** Enterprise-grade security standards for handling sensitive corporate data.

## **6\. Integrations & Developer Tools**

* **Obsidian Plugin:** Embed and edit drawings directly within your markdown notes.  
* **VS Code Extension:** View and edit .excalidraw files inside your IDE.  
* **NPM Package:** Integrate the Excalidraw whiteboard as a React component into your own web applications.  
* **Browser PWA:** Install as a "Progressive Web App" for offline use and desktop-like performance.
